- [ ] 시작 메시지 출력

- [ ] 코치 이름 입력

  - [ ] 코치 5명 이하인지 검증
  - [ ] 잘못된 입력시 에러 메시지 출력 및 재시도

  - [ ] 잘못된 입력시 에러 메시지 출력 및 재시도
